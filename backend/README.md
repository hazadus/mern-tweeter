# MERN-Twatter Backend

## Notes

- Use `nodemon` to restart script when code changed and saved:

```bash
# Globally install nodemon:
npm install -g nodemon
# Then:
nodemon server.js
# or:
npm run dev
```

## Libraries used

- `nodemon`
- `dotenv` - load env vars from `.env` files
- `mongoose`
- `joi`
